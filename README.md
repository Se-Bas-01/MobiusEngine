# Mobius Engine

This engine is a type of library to include in any C++ or C# project

You can use Mobius Engine to easily make 2D games in C++ or C#.

In just 20 lines of code, you can have a simple window and player image.

Would recommend using software like CLion on Mac or Visual Studio on Windows to make the CMake Project template.

Clion - Mac: [https://www.jetbrains.com/clion/download/?section=mac]
